1. Install flask using pip install flask 2.0.0.   This is what I used.
2. Install Werkzeug 2.0.0 using pip install Werkzeug.
3. Install Postman from https://www.postman.com/ to interact with the blockchain
4. Make sure the code is executed from File Explorer.
5. Copy the address shown in the output terminal after execution on the postman get function( example http://172.16.214.31:5000/)
6. http://172.16.214.31:5000/get_chain -> Shows the current node
7. http://172.16.214.31:5000/mine_block -> mines the block with consensus.
8. Click enter.
